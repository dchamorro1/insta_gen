# insta_gen

Generate shareable images that pair an **AI-generated background** with a
**randomly selected verse of the Qur'an**, rendered with a custom Arabic font.

The verse text and metadata come from the public
[Al-Quran Cloud API](https://alquran.cloud/api). By default the background is
generated by a **two-step AI pipeline** so it reflects what the verse is about:

```
verse → (chat model writes a scene description) → (image model renders it) → overlay text
```

You can also supply your own background image and skip generation entirely.

![Sample output](docs/sample_output.png)

## Features

- Fetches a uniformly random ayah (Arabic + English translation + citation) in a
  single API call.
- Interprets the verse with a chat model into a vivid, verse-specific image
  prompt, then renders it with the OpenAI image API. The model picks the style
  that best fits each verse. People and figures may appear, but both stages are
  constrained to keep faces blank, obscured, or silhouetted — consistent with
  Islamic aniconism.
- Composes wrapped, centred, outlined text onto the image with Pillow.
- Graceful degradation: if prompt composition fails, it falls back to a generic
  prompt rather than aborting.
- Reproducible verse selection via `--seed`.

## Architecture

The project is a small package of single-responsibility modules, so each
external dependency sits behind a narrow, testable boundary:

| Module | Responsibility |
| ------ | -------------- |
| `config.py`   | Environment-driven settings (`Settings`) |
| `quran.py`    | `QuranClient` — typed wrapper over the Al-Quran Cloud REST API, returning a `Verse` |
| `prompt.py`   | `VersePromptComposer` — chat model turns a verse into an image prompt |
| `images.py`   | `ImageGenerator` — background generation via OpenAI + download |
| `overlay.py`  | Pure text-wrapping + Pillow rendering of a verse onto an image |
| `pipeline.py` | `create_post()` — orchestrates fetch → compose → generate → render → save |
| `__main__.py` | Argparse CLI (`python -m insta_gen` / `insta-gen`) |

## Setup

```bash
python -m venv .venv && source .venv/bin/activate
pip install -e .            # or: pip install -r requirements.txt
cp .env.example .env        # add your OPENAI_API_KEY (only needed to generate backgrounds)
```

## Usage

Pick a random verse, let a chat model write a scene description for it, generate
that background with OpenAI, and overlay the verse:

```bash
insta-gen --verbose
```

Use your own background image (no API key required):

```bash
insta-gen --background path/to/image.png --seed 42
```

Override the auto-composed prompt with a fixed one:

```bash
insta-gen --prompt "Soft dawn light over still desert dunes, no living beings"
```

Common options:

```
-b, --background PATH   Use an existing image instead of generating one
-p, --prompt TEXT       Override the auto-composed, verse-specific prompt
-o, --output-dir DIR    Where to write the result (default: ./generated_images)
    --seed N            Reproducible verse selection
-v, --verbose           Info-level logging
```

The result is written to `generated_images/verse_<id>.png` and the chosen
verse is printed to stdout.

## Development

```bash
pip install -e ".[dev]"
pytest
```

`tests/` covers the pure rendering/wrapping logic, which needs neither network
access nor an API key.

## Notes & limitations

- The **English translation** is rendered onto the image in DejaVu Sans for
  legibility. The bundled Aalmaghribi calligraphy font is reserved for Arabic.
- Only the translation is composited today. Correct right-to-left Arabic
  shaping in Pillow requires the optional `libraqm` system library; the Arabic
  text is still fetched and printed by the CLI in the meantime.
- The Al-Quran Cloud API is public and unauthenticated; be mindful of its rate
  limits for bulk use.
